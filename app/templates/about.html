About

A minimal web application with a Flask backend and Jinja2 server-side rendering.

This stack is ideal for content-focused sites, dashboards, or internal tools where you want full control over HTML and minimal JavaScript.

  • Flask — WSGI app and API routes
  • Jinja2 — Templates with inheritance, blocks, and filters
  • Static assets — CSS (and optional JS) in app/static
